{{% notice content "前述" %}}

今天一早与 qq 和高兴见面，我们简单逛了一下故宫，中午在紫光园一起吃饭，最后回海淀校区，等到下午四点半的校车坐回丰台。

我非常开心能见到高兴，难得他千里迢迢赶来一次北京，希望他玩得开心，也祝福他找到心仪的工作。而我已经逐渐习惯 ETHPanda 和 LXDAO 共同举办的 Web3 实习计划的学习节奏，包括每日的分享会和笔记学习。但这学期所在校区终究还是太偏僻，要与朋友见面和叙旧，怎么也得花上一个早上的时间，回到学校后已是晚上，能学习的时间少了许多。因而我今天并没有记录许多内容，只是参加了第一次学员的分享例会。

{{% /notice %}}


今晚的分享会给我的感受简直是「八仙过海，各显神通」，又或者像其他学员说的那样「误闯天家」。无论是运营向还是技术向的同学都很有自己的想法——根据自己的想法，有运营向的学员已经在 X 上差不多起了账号，而技术向学员中也有人已经做出了可以运行的钱包 demo，有的学员只是分享自己的学习方法和思路，但也足够让我感到震惊。钦佩之余，我内心也感到有些沉重——好吧，有种前两天好不容易才建立的「心理防线」又被击溃的感觉。但我好好想了一下，还是觉得学习到的内容显然不止 Web3 领域，还有大家如何理解与处理知识。

## Howard

我来晚了一些，没能听完 Howard 的完整分享，但是在 Howard 的分享里听出他很有激情，开了一个好头。

大家都直呼 Howard 是 J 人。

## Vincent

Vincent 借助 Figma 思维导图软件来组织自己对实习手册中关于 Web3 的介绍的理解。

讲的内容简洁，但是很精炼，也很清晰。

## Swen

Swen 自述经济学出身，辞去了一份稳定和收入不错的工作，追求古朴简约。

他的记录似乎也很有意思，在看了相关的采访视频后，以一个一个提点的方式来分享自己的感想，有点像是以太坊中文周会的形式。而且他的叙述很平静，感觉很稳。

我觉得这种形式很有意思，相对于我过去喜欢「复现」学习内容的笔记记录方式，这种形式带来了一种新的想象：分散、更突出和强调个人的所思所想。

## 周星星

周星星分享的是自己的心路，走心局。按照自己的节奏来走吧，走下去。

## Max

Max 分享的是技术向的内容，分享内容里代码扑面而来，看起来十分硬核，主要介绍了 Uniswap。在 GitHub 里，Uniswap 有两个仓库，一个面向用户。

在 Max 的代码分享中，看起来 Web3 技术向对于区块链中的交易原理理解要求会比较高。

Max 对代码的解释很熟练、很具体也很稳，在ETHPanda 说到这些是技术进阶的内容。不过我基本是一头雾水，佩服 Max。

## 奕铭

奕铭讲的内容是运营向，分享的是学习的过程，关于如何输入和输出知识。这给我的启示还是要进行费曼学习法，但其实费曼学习法不一定要面向他人，也可以面向 AI。或者说以能让别人理解为目标来做笔记。

我可以试着将一周的内容总结一下，然后再到 X 上分布。

舍弃完美主义，选择迭代思维。

## Karen

Karen 是中文系的研究生，分享的内容是自己起号的经验。

观察 DAN KOE 的推特文章特点——标题党，配图吸引人。

起号还要厚脸皮，例如可以多点引用别人的内容。做自己的水军。

在 X 上表演，将自己推销出去。

我感觉 Karen 将自己的中文系写作水平在 X 的营销中发挥出来了。但是她本身也有着很强的观察能力，例如刷了两个多月的推特，从兴趣入手，观察、总结别人在写什么，怎么写，之后再输出。

Karen 也分享了具体的写作方法，如果要高效整理自己的思考，可以从写是什么，到为什么，再到怎么做；而如果要把文章写得有故事性，可以从起因、经过、结果等简单因素出发。

## 小白

小白分享 solidity 语法的学习。看起来 solidity 和 python 相近。小白最后说的找到适合自己做的很有意思，做某件事情如果无法乐在其中，可以考虑换一个赛道。例如如果只是为了流量而做流量，那也是很难走远。小白说得很朴素，但不知为什么就很打动我，能感觉到他的实诚。当然，小白自身的生活习惯也很惊人，早上五点起床跑个五公里，然后七点开始学习，中午睡个午觉后，下午一点多学习到五点，然后吃晚饭……当然，我不会模仿小白，毕竟每个人的习惯、方式和节奏都不一样。看到大家都那么优秀，我内心很是动摇，但小白的分享却又让我感到想明白自己的赛道，或者说做自己乐意和愿意坚持下去的事情很重要。

这才第一周，huahua 直接做了一个 MPC 钱包的 demo，并且部署好了在网站上。我直接被震惊。

## Leon

Leon 也是通过思维导图的方式来分享自己对 Web3 的学习。如果只是读笔记，其实还是很机械的学习方式，但是通过思维导图的打开和关闭过程，可以以此来进行检测自己。

学习不要只是局限在笔记里。可以扩展开来。做笔记的目标不是仅仅复现，而是能做到吸收知识。
