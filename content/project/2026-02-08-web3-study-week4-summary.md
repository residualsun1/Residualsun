今天已经是实习计划的最后一天。28 天残酷共学，可以说是真正的「残酷」共学——如果只是查看共学的[面板数据](https://intensivecolearn.ing/programs/Web3_Internship_Bootcamp_2026_Winter)，会发现 500 人左右报的名，最后也不过二十来人完成了全程「打卡」。虽然打卡并不是有效学习的最好证明，但它可以从侧面反映出共学的参与情况。另外，都说养成一个习惯最少都需要 21 天，相比于动辄 3-4 个月的传统实习，28 天的 Web3 实习计划虽然看起来不算长，但认真实习并不一定能真的学习到知识——因为可能只是作为牛马进行重复「劳作」，而 Web3 实习却实打实给我带来了成长，我多了一份完全不同的学习体验，对未来道路的认识虽谈不上变得清晰，但又好像看到了新的方向。

在研一第一学期的末尾，我非常清晰且具体地认识到**我不要读博深造**，我想在剩下的一段时间里——研一第二学期、研二与研三——弄清楚：**我要学习怎样的技术，是 Web2 还是 Web3？我现在可以尝试探索怎样的职业方向，是 UI 设计师还是产品经理？又或者是更多的其他什么，比如……与 AI 有关的，甚至就如 wachi 老师对我说的，技术好奇心与人文知识的结合可以做出一些很好玩的东西**？[^human]

[^human]: 在这次休闲黑客松中，我感受到自己的人文思维在产品叙事与路演准备中发挥了相当程度的作用。

在我念本科的时候，从 R 语言开始，我应当算是接触了计算机科学这一领域——这可能是一个很抽象、狭隘又很局限的说法，因为它蕴含的是我这个所谓的文科生[^word]对于理科生的「所有」想象。某种程度上来说，对于「编程」，我抱有某种「拜物」般的情结。后来，由于学习 R 语言需要到相关论坛求助，我学会了翻墙，翻墙以后了解到 GitHub，接着开始搭建个人网站，其间断断续续地学习了一些 HTML 与 CSS，而为了美化网站，间或又零零碎碎地积累了一些其他的具体技术知识，特别是 AI，就此也算是开始徘徊于「技术」的大门之外。

[^word]: 这种「文科生」的言辞往往带有某种认为文科知识学习者与技术无缘的深刻偏见。

必须承认，AI 发展得十分迅猛，这或许会让不少人为此感到惊恐，但细细一想，至少作为普通人兼所谓文科生的我却感受到了许多由其带来的红利——ETHPanda 和 LXDAO 联合发起的 2026 Web3 冬季实习计划便是我近期体验与学习 AI 最好的学习平台。不过需要澄清的是，这里的 AI 并非单纯指 AI 领域中的具体知识，也包括了与 AI 交互以学习其他领域知识的这一过程。没有冬季实习计划，我便不知道要用 AI 来对哪个领域的知识进行学习，更不知道有哪些强大的 AI 工具可用于进行开发。ETHPanda 和 LXDAO 提供了一个系统的平台和学习周期。

如果要用一句话概括我对 2026 Web3 冬季实习计划的感受，我最想说的是「**这四周走来，一路没有敌人，全是老师**」。我会记住很多人——第一次让我感受到 Web3 是一个理想、以建设作为愿景的 Bruce 老师，开口即是金句、看起来很是闲云野鹤但实则深藏功与名的超级大佬 wachi 老师，开营期间便给同学写了许多任务详细教程的闷骚森理，自律到让我侧目的高执行力者兼我的休闲黑客松队友小白，被称作励志姐并坚持不懈在 X 上发文运营，最终也成功拿到 offer 的 Karen，做出「Wit Or Die」项目与「EIP Playground」的项目团队成员……我在实习计划中，个人通过课程安排教学所汲取的知识并不算多，反而是在和助教与学员的交流分享中收获最多。

因为此次计划的学习性质、模式与工具的要求，我「翻墙」的程度比过往更深——不认真翻不知道，翻了才知道还有一番讲究。除了 Telegram 我还无法注册，我终于迈出了过去难以在 iPhone 上获取梯子的难题——我在一位学员分享的梯子文档中找到了可共享的 Apple Store 海外账号，由此得以在内地通过 iPhone 而不再只是电脑浏览 X，收听 Spotify，以及随时使用 Gemini。而这段时间对 X 的使用体验告诉我，如果要了解 Web3 或 AI，X 一定是一个国内社媒难以取代的重要平台。

当然，这只是开始。在实习计划中，同期计划的学员们在每周例会中分享自己的学习心得、方法，我由此得以了解原来 Vibe Coding 可以通过 Antigravity、Cursor、OpenCode 这样的 IDE 来完成，随之而来的第三周休闲黑客松更是「逼」得我不得不对此进行学习和尝试。我既在 X 上不断浏览 Web2 与 Web3 领域的爱好者或从业人员如何分享他们关于 Vibe Coding 的想法和体验，也摸爬滚打地在 Antigravity 中笨拙地进行尝试。在技术知识十分匮乏的情况下，我第一次 Vibe Coding 时全程托管 AI，第一次使用 Claude Opus 4.5 便耗尽了 Tokens，但也震惊于它是如何快速高效地完成了一项简单的 DApp 开发——我感受到了 AI 的强大，虽然这只是开始，虽然这背后其实也隐含着某种未及浮现的「危险」[^danger]。

[^danger]: 这里的「危险」在《[Web3 实习计划（冬季）：技术 / 开发向的成长路径](https://www.bilibili.com/video/BV1xzFez9EYx/?share_source=copy_web&vd_source=3f89400ae195feff9dd7d805682d7814)》中得到了具体的阐述。

事情并没有就此结束。由于 Antigravity 中的模型调用 Tokens 有限，而我还是技术小白，很难把控与 AI 交流的程度，因而时时面临着 Tokens 耗尽的「危机」，不得不到网络寻找新的方案，继而发现可以在计算机终端运行的 Claude Code 和 Codex——我在浏览器与 X 上都见过。调用这些模型需要使用海外信用卡进行订阅，这又是一个问题，所幸还有「中转站」——这也是我兜兜转转之下了解到的途径。经由中转站，我可以使用内地的支付方式，花费比直接订阅更低的金额来获取 Claude Code 和 Codex 的使用 Tokens。

我本以为自己会高频使用 Antigravity，但在休闲黑客松期间，由于 Codex 的使用不限限额，我反而更习惯在终端进行 Vibe Coding。一次又一次与 AI 交互，我本来对终端代码的书写一直抱有一种恐惧感，但都在 Coding 的过程中变得越来越熟悉，甚至感到亲近。而在具体的项目里，我又遭遇了如何通过 Python 调用大模型 API 的问题，了解应该怎么运行一个普通的脚本……

说起来其实很惭愧，本次实习计划的初衷是为 Web3 领域行业培养高质量的实习生与从业人员，但我在 Web3 领域知识的学习并不深入，也不优秀，甚至不知晓是否算作打下了基础，但真的很感谢此次实习计划，让我不仅能对 Web3 形成一定程度的了解，也对 AI 产生了更多认识——最让我开心的是我借此知道自己也可以 Vibe Coding，这对我来说是一种无比宝贵的学习方式。回想 4 周以前我是如何迷茫焦虑，我感觉面前的知识很是陌生，自己又好像真的很难对它们提起足够的兴趣，但是密集的知识分享却实实在在地给到我成长和收获。我只能说怪自己不争气，只能在最低程度上吸取到知识。

我甚至在此次实习计划中成功参加与完成了一次 Web3 黑客松。在我们的「时间赌徒」项目中，队友小白主要负责整个项目的代码 Vibe Coding，我在此基础上进行细微的调整，提供 UI 修改建议，构思产品理念与叙事，最后完成路演。出乎我意料的是，这虽然都是我们第一次参加黑客松，项目却还是在我们选择的 LLM 赛道中被评为亚军。黑客松拿奖对我的激励很大，但与奖金无关，纯粹是因为这被我视为评委对我们 idea 的认可，甚至是「看见」了我的叙事——在我们的整个项目之中，小白的技术十分重要，没有他先做出的 Demo，我们也不知道沿着什么具体方向修改完善，但我认为自己阐述的产品理念与路演在 5 分钟内也发挥了关键作用，我似乎在这里面能感受出将技术与人文结合的尝试，而正因为技术与人文的结合，所以我们的产品才能在与其他技术栈和 UI 设计都十分优秀的产品的竞争中拿到了亚军。

而同等重要，甚至更为重要的是，我深刻感受到了「不同的世界」，看见了许多优秀的学员——他们的想法和行动都让我学到了很多，而 wachi 老师说的，「**每一个小小的任务，每一次发言，每一个交流都是在增加自己的 credit。让别人知道自己是一个什么样的人，能做什么样的事，价值观是什么样的**」将一直一直留在我心里，激励我——虽然我一直没能做到。我会想起自己看着学员们主动分享自己的技术知识或者个人思考时是如何慢慢被他们吸引，在这看起来漫长而知识密集的实习计划期间，我一度感到枯燥和难耐，但也正是在这样一个学习平台里，我在和休闲黑客松的队友们的讨论中会感受到跃动的心情，最后进而想到 ETHPanda 和 LXDAO 的包容、开放、公共、开源精神是多么可贵，完全免费地为我们这些背景各异的学员提供诚意满满、系统性的学习资源。

从一开始，其实我便没有抱着如何明确的目的进入实习计划，因而最后没有找到相应的 Web3 实习或工作岗位，不过我并不为此感到十分气馁。说起来，我感受到更多的其实是幸运，这里的幸运，具体指的是我总作为「受赠」却又不被明确与严格要求进行回报的角色而穿梭于生活洪流之中。我认为此次实习计划仍然是期望能培养出高质量的 Web3 领域运营者与开发者，但我两头都没占，就此而言还是觉得没能「对得起」ETHPanda 和 LXDAO 在几千份的报名中给予了录取我为学员的机会；但通过实习计划，我关于自己未来想象的空间得到了扩展——我也可以尝试 Vibe Coding，我体验过给一个产品提出 UI 设计的建议，我为一个项目进行过以叙事为主导的路演（还夹带了一点自己的人类学私货）…… 

最后，我认为自己还有许多话要说，但又不必急于一时全部说完，也无法于此时此刻就倾尽所有，那么就在此留白，为以后的书写留白。
