## 2. 首行缩进

在文本前添加以下字符。

```
&emsp;
```

示例如下：

&emsp;莫斯的《礼物》是一本既很容易读，也很难读的书。

&emsp;&emsp;莫斯的《礼物》是一本既很容易读，也很难读的书。

第一句话句首加了`&emsp;`，第二句加了`&emsp;&emsp;`
